# UXAudit 🚀

A modern UX Audit web application built using React, TypeScript, Vite, and Tailwind CSS.  
It allows users to analyze website UX and view insights in a clean dashboard.

## 🌐 Live Demo
Project URL: https://ux-audit-new.vercel.app/

## ⚙️ Tech Stack
- React + TypeScript
- Vite
- Tailwind CSS
- Vercel (Deployment)
